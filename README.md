# 📄 RAG Document Q&A System

## 📌 Overview
The RAG Document Q&A System is an AI-powered application that enables users to upload PDF and DOCX documents and ask questions in natural language. The system retrieves relevant information from the document and generates context-aware answers using Large Language Models (LLMs).

## 🚀 Features
- Upload PDF and DOCX files
- Ask questions in natural language
- Semantic search and document retrieval
- Context-aware answer generation
- User-friendly Streamlit interface

## 🛠️ Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- Large Language Models (LLMs)
- Supabase
